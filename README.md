## Come attivare GitHub Pages

1. Crea un nuovo repository (es. `segnale-e-rumore`)
2. Carica tutti i file di questa cartella nella root del repo
3. Vai su **Settings → Pages**
4. Source: **Deploy from a branch** → branch `main` → cartella `/ (root)`
5. Clicca **Save**

La demo sarà disponibile su:
`https://TUO-USERNAME.github.io/segnale-e-rumore/`
